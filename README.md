# MineraBroadCastUI
MineraBroadCastUI is an useful plugin which you can send broadcast messages.

## Commands + Permission
Command | Description | Default | Alias | Permissions
--------- | ------------------- | ------- | ---------- | -----------
``/broadcast`` | ``Sending a broadcast message`` | ``op`` | ``bc`` | ``minerabroadcastui.broadcast.cmd``
